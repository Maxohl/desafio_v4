# Desafio v4

## Api RESTful 
Nesse desafio foi desenvolvido uma API RESTful simples.

## Objetivo do Desafio
Criar uma API assíncrona de uma academia para competição crossfit. Usando o framekwork fastAPI.

Detalhes do desafio:
 - O desafio tem 3 tabelas, Atleta, Categoria e Centro de Treinamento.
 - Atleta tem relação 1:1 com as duas outras tabelas.
 - Criar as rotas para pegar, inserir, editar e deletar os dados da tablea.
 - Adicionar query parametros nos endpoints.
 - Personalizar resposta de retorno nos endpoints.
 - Tratar exceção de integridade dos dados de cada módulo/tabela.
 - Adicionar paginação com a biblioteca fastapi-pagination
